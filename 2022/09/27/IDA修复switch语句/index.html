  2. Number of Elements: 数组元素个数

  4. Element base value: offset ，也就是每个元素要加的base，从汇编代码可以看出来就是jump table的地址
  5. Start of the switch idiom: switch开始的地方
  6. Input register of switch : 被当作idx的寄存器
